# Project Title 

Automation testing for Octopus insight page  

## Getting Started

In order to use this code make sure you are using a Maven project and update your Pom.xml file with the required dependencies mentioned in project Pom file.

### Prerequisites

Must use a IDE i.e. Intellij or Eclipse 

## Running the tests

To run the test in local machine clone the repositry from git, once downloaded open it in a IDE and run Testrunner.java


## Built With

* BDD,Junit, Selenium Webdriver with Java - The web framework used
* [Maven](https://maven.apache.org/) - Dependency Management
* Java, Cucumber/Gherkin - Languages used for coding
